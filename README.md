# Jigger
🍸 App PWA para bartenders profesionales — 77 cócteles IBA + carta de autor propia

JIGGER es una herramienta gratuita para bartenders profesionales. 
Identifica cócteles clásicos IBA por ingredientes, permite crear 
tu propia carta de autor con o sin medidas, y en el futuro será 
una comunidad donde los bartenders comparten sus creaciones.
